# aaa_result_info


Provides information about the result of the request, including count and cursor.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `count` | No | string | The number of records returned in the response. |
| `cursor` | No | string | The cursor token used for pagination. |
