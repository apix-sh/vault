# api-shield_parameter_schemas_definition


An operation schema object containing a response.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `parameters` | No | array<object> | An array containing the learned parameter schemas. |
| `responses` | No | object | An empty response object. This field is required to yield a valid operation schema. |
