# dos_Prefix

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `comment` | Yes | string | A comment describing the prefix. |
| `created_on` | Yes | string | The creation timestamp of the prefix. |
| `excluded` | Yes | boolean | Whether to exclude the prefix from protection. |
| `id` | Yes | string | The unique ID of the prefix. |
| `modified_on` | Yes | string | The last modification timestamp of the prefix. |
| `prefix` | Yes | string | The prefix in CIDR format. |
