# firewall_asn_configuration

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `target` | No | string | The configuration target. You must set the target to `asn` when specifying an Autonomous System Number (ASN) in the rule. Allowed values: asn |
| `value` | No | string | The AS number to match. |
