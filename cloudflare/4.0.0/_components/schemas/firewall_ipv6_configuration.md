# firewall_ipv6_configuration

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `target` | No | string | The configuration target. You must set the target to `ip6` when specifying an IPv6 address in the rule. Allowed values: ip6 |
| `value` | No | string | The IPv6 address to match. |
