# firewall_ua_configuration

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `target` | No | string | The configuration target. You must set the target to `ua` when specifying a user agent in the rule. Allowed values: ua |
| `value` | No | string | the user agent to exactly match |
