# hyperdrive_internet-origin

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `host` | Yes | string | Defines the host (hostname or IP) of your origin database. |
| `port` | Yes | integer | Defines the port of your origin database. Defaults to 5432 for PostgreSQL or 3306 for MySQL if not specified. |
