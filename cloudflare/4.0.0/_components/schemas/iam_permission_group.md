# iam_permission_group


A named group of permissions that map to a group of operations against resources.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `id` | Yes | string | Identifier of the permission group. |
| `meta` | No | object | Attributes associated to the permission group. |
| `name` | No | string | Name of the permission group. |
