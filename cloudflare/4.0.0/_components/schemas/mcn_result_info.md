# mcn_result_info

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `count` | Yes | integer | The number of items in the current result set. |
| `page` | Yes | integer | The current page (starts from zero). |
| `per_page` | Yes | integer | The maximum number of items per page. |
| `total_count` | Yes | integer | The total number of items in the entire result set. |
| `total_pages` | No | integer | The number of total pages in the entire result set. |
