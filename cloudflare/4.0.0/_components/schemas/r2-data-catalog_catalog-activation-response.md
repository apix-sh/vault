# r2-data-catalog_catalog-activation-response


Contains response from activating an R2 bucket as a catalog.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `id` | Yes | string | Use this to uniquely identify the activated catalog. |
| `name` | Yes | string | Specifies the name of the activated catalog. |
