# realtimekit_Participant


Represents a participant.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `created_at` | Yes | string | When this object was created. The time is returned in ISO format. |
| `custom_participant_id` | Yes | string | A unique participant ID generated by the client. |
| `id` | Yes | string | ID of the participant. |
| `name` | No | string | Name of the participant. |
| `picture` | No | string | URL to a picture of the participant. |
| `preset_name` | Yes | string | Preset applied to the participant. |
| `updated_at` | Yes | string | When this object was updated. The time is returned in ISO format. |
