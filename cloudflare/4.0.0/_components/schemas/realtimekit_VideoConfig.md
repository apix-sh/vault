# realtimekit_VideoConfig

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `codec` | No | string | Codec using which the recording will be encoded. Allowed values: H264, VP8 |
| `export_file` | No | boolean | Controls whether to export video file seperately |
| `height` | No | integer | Height of the recording video in pixels |
| `watermark` | No | object | Watermark to be added to the recording |
| `width` | No | integer | Width of the recording video in pixels |
