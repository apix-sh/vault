# rulesets_RedirectFromList


A redirect based on a bulk list lookup.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `key` | Yes | string | An expression that evaluates to the list lookup key. |
| `name` | Yes | string | The name of the list to match against. |
