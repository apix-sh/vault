# rulesets_SetConfigAutominify


Which file extensions to minify automatically.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `css` | No | boolean | Whether to minify CSS files. |
| `html` | No | boolean | Whether to minify HTML files. |
| `js` | No | boolean | Whether to minify JavaScript files. |
