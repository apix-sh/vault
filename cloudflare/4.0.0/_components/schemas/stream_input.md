# stream_input

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `height` | No | integer | The video height in pixels. A value of `-1` means the height is unknown. The value becomes available after the upload and before the video is ready. |
| `width` | No | integer | The video width in pixels. A value of `-1` means the width is unknown. The value becomes available after the upload and before the video is ready. |
