# web3_result_info

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `count` | No | number | Specifies the total number of results for the requested service. |
| `page` | No | number | Specifies the current page within paginated list of results. |
| `per_page` | No | number | Specifies the number of results per page of results. |
| `total_count` | No | number | Specifies the total results available without any search parameters. |
