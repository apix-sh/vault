# workers_ErrorWorkerNameSubdomainLengthLimit

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `code` | Yes | integer | Code indicating that the Worker name is too long to be used as a subdomain. |
