# workers_ErrorWorkerTagLimit

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `code` | Yes | integer | Code indicating that the Worker has exceeded the maximum number of tags allowed. |
| `message` | Yes | string | Message explaining that the tag limit has been exceeded and suggesting to remove a tag. |
