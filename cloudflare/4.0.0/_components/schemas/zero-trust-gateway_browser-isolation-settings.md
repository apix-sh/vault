# zero-trust-gateway_browser-isolation-settings


Specify Clientless Browser Isolation settings.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `non_identity_enabled` | No | boolean | Specify whether to enable non-identity onramp support for Browser Isolation. |
| `url_browser_isolation_enabled` | No | boolean | Specify whether to enable Clientless Browser Isolation. |
