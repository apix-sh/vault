# zero-trust-gateway_result_info

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `count` | No | number | Indicate the total number of results for the requested service. |
| `page` | No | number | Indicate the current page within a paginated list of results. |
| `per_page` | No | number | Indicate the number of results per page. |
| `total_count` | No | number | Indicate the total results available without any search parameters. |
