# zero-trust-gateway_sandbox


Specify whether to enable the sandbox.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `enabled` | No | boolean | Specify whether to enable the sandbox. |
| `fallback_action` | No | string | Specify the action to take when the system cannot scan the file. Allowed values: allow, block |
