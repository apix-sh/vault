# zone-analytics-api_pageviews


Breakdown of totals for pageviews.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `all` | No | integer | The total number of pageviews served within the time range. |
| `search_engine` | No | object | A variable list of key/value pairs representing the search engine and number of hits. |
