# actions-cache-retention-limit-for-organization


GitHub Actions cache retention policy for an organization.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `max_cache_retention_days` | No | integer | For repositories in this organization, the maximum duration, in days, for which caches in a repository may be retained. |
