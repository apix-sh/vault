# actions-cache-storage-limit-for-organization


GitHub Actions cache storage policy for an organization.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `max_cache_size_gb` | No | integer | For repositories in the organization, the maximum size limit for the sum of all caches in a repository, in gigabytes. |
