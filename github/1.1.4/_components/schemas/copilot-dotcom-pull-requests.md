# copilot-dotcom-pull-requests


Usage metrics for Copilot for pull requests.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `total_engaged_users` | No | integer | The number of users who used Copilot for Pull Requests on github.com to generate a pull request summary at least once. |
| `repositories` | No | array<object> | Repositories in which users used Copilot for Pull Requests to generate pull request summaries |
