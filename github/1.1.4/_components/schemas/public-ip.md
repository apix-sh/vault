# public-ip


Provides details of Public IP for a GitHub-hosted larger runners

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `enabled` | No | boolean | Whether public IP is enabled. |
| `prefix` | No | string | The prefix for the public IP. |
| `length` | No | integer | The length of the IP prefix. |
