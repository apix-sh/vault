# actions-cache-retention-limit-for-enterprise


GitHub Actions cache retention policy for an enterprise.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `max_cache_retention_days` | No | integer | For repositories & organizations in an enterprise, the maximum duration, in days, for which caches in a repository may be retained. |
