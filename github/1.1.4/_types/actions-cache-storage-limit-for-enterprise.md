# actions-cache-storage-limit-for-enterprise


GitHub Actions cache storage policy for an enterprise.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `max_cache_size_gb` | No | integer | For repositories & organizations in an enterprise, the maximum size limit for the sum of all caches in a repository, in gigabytes. |
