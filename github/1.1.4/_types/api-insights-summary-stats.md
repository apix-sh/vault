API Insights usage summary stats for an organization

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `total_request_count` | No | integer | The total number of requests within the queried time period |
| `rate_limited_request_count` | No | integer | The total number of requests that were rate limited within the queried time period |
