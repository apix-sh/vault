# check-automated-security-fixes


Check Dependabot security updates

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `enabled` | Yes | boolean | Whether Dependabot security updates are enabled for the repository. |
| `paused` | Yes | boolean | Whether Dependabot security updates are paused for the repository. |
