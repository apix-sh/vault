# copilot-dotcom-chat


Usage metrics for Copilot Chat in GitHub.com

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `total_engaged_users` | No | integer | Total number of users who prompted Copilot Chat on github.com at least once. |
| `models` | No | array<object> | List of model metrics for a custom models and the default model. |
