# dependabot-public-key


The public key used for setting Dependabot Secrets.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `key_id` | Yes | string | The identifier for the key. |
| `key` | Yes | string | The Base64 encoded public key. |
