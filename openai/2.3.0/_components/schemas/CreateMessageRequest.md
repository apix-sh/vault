# CreateMessageRequest

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `role` | Yes | string | The role of the entity that is creating the message. Allowed values include:
- `user`: Indicates the message is sent by an actual user and should be used in most cases to represent user-generated messages.
- `assistant`: Indicates the message is generated by the assistant. Use this value to insert messages from the assistant into the conversation.
 Allowed values: user, assistant |
| `content` | Yes | oneOf(2) |  |
| `attachments` | No | any | A list of files attached to the message, and the tools they should be added to. |
| `metadata` | No | [Metadata](Metadata.md) |  |
