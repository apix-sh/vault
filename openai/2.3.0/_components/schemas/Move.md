# Move


A mouse move action.


## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `type` | Yes | string | Specifies the event type. For a move action, this property is 
always set to `move`.
 Allowed values: move |
| `x` | Yes | integer | The x-coordinate to move to.
 |
| `y` | Yes | integer | The y-coordinate to move to.
 |
