# Image


Represents the content or the URL of an image generated by the OpenAI API.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `b64_json` | No | string | The base64-encoded JSON of the generated image. Default value for `gpt-image-1`, and only present if `response_format` is set to `b64_json` for `dall-e-2` and `dall-e-3`. |
| `url` | No | string | When using `dall-e-2` or `dall-e-3`, the URL of the generated image if `response_format` is set to `url` (default value). Unsupported for `gpt-image-1`. |
| `revised_prompt` | No | string | For `dall-e-3` only, the revised prompt that was used to generate the image. |
