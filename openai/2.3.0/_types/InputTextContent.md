# InputTextContent


A text input to the model.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `type` | Yes | string | The type of the input item. Always `input_text`. Allowed values: input_text |
| `text` | Yes | string | The text input to the model. |
