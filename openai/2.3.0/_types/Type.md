# Type


An action to type in text.


## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `type` | Yes | string | Specifies the event type. For a type action, this property is 
always set to `type`.
 Allowed values: type |
| `text` | Yes | string | The text to type.
 |
