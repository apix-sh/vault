# account_dashboard_settings

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `display_name` | No | string | The display name for this account. This is used on the Stripe Dashboard to differentiate between accounts. |
| `timezone` | No | string | The timezone used in the Stripe Dashboard for this account. A list of possible time zone values is maintained at the [IANA Time Zone Database](http://www.iana.org/time-zones). |
