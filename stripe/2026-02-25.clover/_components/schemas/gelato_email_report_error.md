# gelato_email_report_error

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `code` | No | string | A short machine-readable string giving the reason for the verification failure. Allowed values: email_unverified_other, email_verification_declined |
| `reason` | No | string | A human-readable message giving the reason for the failure. These messages can be shown to your users. |
