# insights_resources_payment_evaluation_money_movement_card


Money Movement card details attached to this payment.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `customer_presence` | No | string | Describes the presence of the customer during the payment. Allowed values: off_session, on_session |
| `payment_type` | No | string | Describes the type of payment. Allowed values: one_off, recurring, setup_one_off, setup_recurring |
