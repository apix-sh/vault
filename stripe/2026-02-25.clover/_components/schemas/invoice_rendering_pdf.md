# invoice_rendering_pdf

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `page_size` | No | string | Page size of invoice pdf. Options include a4, letter, and auto. If set to auto, page size will be switched to a4 or letter based on customer locale. Allowed values: a4, auto, letter |
