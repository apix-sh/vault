# issuing_physical_bundle_features

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `card_logo` | Yes | string | The policy for how to use card logo images in a card design with this physical bundle. Allowed values: optional, required, unsupported |
| `carrier_text` | Yes | string | The policy for how to use carrier letter text in a card design with this physical bundle. Allowed values: optional, required, unsupported |
| `second_line` | Yes | string | The policy for how to use a second line on a card with this physical bundle. Allowed values: optional, required, unsupported |
