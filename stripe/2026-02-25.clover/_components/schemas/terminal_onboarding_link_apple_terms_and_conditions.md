# terminal_onboarding_link_apple_terms_and_conditions


Options associated with the Apple Terms and Conditions link type.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `allow_relinking` | No | boolean | Whether the link should also support users relinking their Apple account. |
| `merchant_display_name` | Yes | string | The business name of the merchant accepting Apple's Terms and Conditions. |
