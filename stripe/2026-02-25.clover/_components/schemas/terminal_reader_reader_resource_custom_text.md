# terminal_reader_reader_resource_custom_text


Represents custom text to be displayed when collecting the input using a reader

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `description` | No | string | Customize the default description for this input |
| `skip_button` | No | string | Customize the default label for this input's skip button |
| `submit_button` | No | string | Customize the default label for this input's submit button |
| `title` | No | string | Customize the default title for this input |
