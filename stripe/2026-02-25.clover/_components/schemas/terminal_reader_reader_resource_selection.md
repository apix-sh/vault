| `choices` | Yes | array<[terminal_reader_reader_resource_choice](./terminal_reader_reader_resource_choice.md)> | List of possible choices to be selected |
| `id` | No | string | The id of the selected choice |
| `text` | No | string | The text of the selected choice |
