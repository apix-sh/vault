# bank_connections_resource_link_account_session_filters

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `account_subcategories` | No | array<string> | Restricts the Session to subcategories of accounts that can be linked. Valid subcategories are: `checking`, `savings`, `mortgage`, `line_of_credit`, `credit_card`. |
| `countries` | No | array<string> | List of countries from which to filter accounts. |
