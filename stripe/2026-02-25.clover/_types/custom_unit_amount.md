# custom_unit_amount

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `maximum` | No | integer | The maximum unit amount the customer can specify for this item. |
| `minimum` | No | integer | The minimum unit amount the customer can specify for this item. Must be at least the minimum charge amount. |
| `preset` | No | integer | The starting unit amount which can be updated by the customer. |
