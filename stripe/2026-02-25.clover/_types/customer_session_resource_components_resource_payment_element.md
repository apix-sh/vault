# customer_session_resource_components_resource_payment_element


This hash contains whether the Payment Element is enabled and the features it supports.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `enabled` | Yes | boolean | Whether the Payment Element is enabled. |
| `features` | No | anyOf(1) | This hash defines whether the Payment Element supports certain features. |
