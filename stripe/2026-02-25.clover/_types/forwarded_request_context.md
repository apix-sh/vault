# forwarded_request_context


Metadata about the forwarded request.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `destination_duration` | Yes | integer | The time it took in milliseconds for the destination endpoint to respond. |
| `destination_ip_address` | Yes | string | The IP address of the destination. |
