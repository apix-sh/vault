# insights_resources_payment_evaluation_customer_details


Customer details attached to this payment evaluation.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `customer` | No | string | The ID of the customer associated with the payment evaluation. |
| `customer_account` | No | string | The ID of the Account representing the customer associated with the payment evaluation. |
| `email` | No | string | The customer's email address. |
| `name` | No | string | The customer's full name or business name. |
| `phone` | No | string | The customer's phone number. |
