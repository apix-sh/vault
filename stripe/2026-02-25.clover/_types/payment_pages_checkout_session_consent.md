# payment_pages_checkout_session_consent

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `promotions` | No | string | If `opt_in`, the customer consents to receiving promotional communications
from the merchant about this Checkout Session. Allowed values: opt_in, opt_out |
| `terms_of_service` | No | string | If `accepted`, the customer in this Checkout Session has agreed to the merchant's terms of service. Allowed values: accepted |
