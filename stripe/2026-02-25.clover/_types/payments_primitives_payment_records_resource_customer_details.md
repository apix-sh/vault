# payments_primitives_payment_records_resource_customer_details


Information about the customer for this payment.

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `customer` | No | string | ID of the Stripe Customer associated with this payment. |
| `email` | No | string | The customer's email address. |
| `name` | No | string | The customer's name. |
| `phone` | No | string | The customer's phone number. |
