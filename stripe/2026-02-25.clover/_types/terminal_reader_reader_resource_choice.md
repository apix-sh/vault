# terminal_reader_reader_resource_choice


Choice to be selected on a Reader

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `id` | No | string | The identifier for the selected choice. Maximum 50 characters. |
| `style` | No | string | The button style for the choice. Can be `primary` or `secondary`. Allowed values: primary, secondary |
| `text` | Yes | string | The text to be selected. Maximum 30 characters. |
