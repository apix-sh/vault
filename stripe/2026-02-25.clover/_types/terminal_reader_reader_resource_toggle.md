# terminal_reader_reader_resource_toggle


Information about an input's toggle

## Properties

| Property | Required | Type | Description |
| :------- | :------: | :--- | :---------- |
| `default_value` | No | string | The toggle's default value. Can be `enabled` or `disabled`. Allowed values: disabled, enabled |
| `description` | No | string | The toggle's description text. Maximum 50 characters. |
| `title` | No | string | The toggle's title text. Maximum 50 characters. |
| `value` | No | string | The toggle's collected value. Can be `enabled` or `disabled`. Allowed values: disabled, enabled |
